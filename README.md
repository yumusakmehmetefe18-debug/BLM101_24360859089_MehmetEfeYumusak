Proje Açıklaması
Bu proje, kullanıcının konsol üzerinden girdiği kişisel bilgileri alarak bu verileri görsel olarak düzenlenmiş statik bir web sitesine dönüştüren bir otomasyon yazılımıdır.

Kodun Ne Yaptığı:
Program çalıştırıldığında kullanıcıya adı, aldığı dersler ve biyografisi gibi sorular sorar.

Girilen bu verileri Python'daki String işleme yeteneklerini kullanarak bir HTML şablonuna yerleştirir.

Sonuç olarak, klasörde basit bir CSS ile renklendirilmiş, başlıkları ve listeleri olan bir index.html dosyası oluşturur.

Algoritma Mantığı ve Teknik Detaylar:

Veri Girişi: input() fonksiyonu ile dinamik veriler toplanır.


Veri İşleme: Kullanıcının virgülle ayırarak girdiği dersler split() metodu ile listeye çevrilir ve döngülerle HTML listesine (<li>) dönüştürülür.


Dosya Yazma: Python'un "File Write" (Dosya Yazma) yetenekleri kullanılarak tüm içerik kalıcı bir dosyaya kaydedilir.


Görselleştirme: HTML koduna gömülü CSS (Internal CSS) kullanılarak sayfanın modern bir görünüme sahip olması sağlanmıştır.

Kurulum ve Çalıştırma
Bilgisayarınızda Python yüklü olduğundan emin olun.

src klasöründeki proje.py dosyasını indirin.

Terminal veya IDE (PyCharm vb.) üzerinden dosyayı çalıştırın.

Program bittikten sonra oluşan index.html dosyasını bir web tarayıcısı ile açın.
